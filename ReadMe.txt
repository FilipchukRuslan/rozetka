Folder contains two main projects. There are serverApp Rozetka_test_exercise.sln and 
rozetka-front folder with angular2App. 
Also there are DAO and Model that are used to get acces to the Database.
To investigate the whole project, follow this steps:
1. Run serverProject. You need to simply build it! 
2. Run angular2Proj. You need to install npm packages, simply by writing command
in console (that launched in directory of this project) "npm i". 
Then write "ng serve".
